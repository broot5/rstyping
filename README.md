# :keyboard: rstyping
typing test program written with rust

## How to Use
```
git clone https://github.com/broot5/rstyping.git
cd rstyping
cargo web start
```

## How to Create a Text File
```
example sentence1
example sentence2.
example sentence3!
```
This is how it's identified by a line.
